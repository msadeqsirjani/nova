## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/msadeqsirjani/nova.git
```

Navigate to the project directory

```bash
cd nova
```

Install dependencies

```bash
pip install -r requirements.txt
```

## 💻 Usage

```bash
python src/main.py
```
